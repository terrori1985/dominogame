# 🎲 Многопользовательское Домино для Telegram

Полноценная игра в домино с реальными игроками из Telegram!

## ✨ Особенности

- ✅ **Пустой стол** - ожидание игроков
- ✅ **Игроки из Telegram** - автоматический вход
- ✅ **Создание игр** с разными настройками
- ✅ **Приглашение друзей** по ссылке или коду
- ✅ **Реальное время** - синхронизация ходов
- ✅ **4 типа камней** и 3 дизайна стола
