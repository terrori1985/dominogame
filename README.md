# 🎲 Domino — Telegram Mini App

Многопользовательская игра домино для Telegram с анимациями, звуками и красивым дизайном в стиле покерного зала.

## 🚀 Деплой на Render

1. **Fork** этот репозиторий на GitHub
2. Зайдите на [render.com](https://render.com) и создайте **New Web Service**
3. Подключите репозиторий
4. Настройки:
   - **Build Command:** `npm install`
   - **Start Command:** `npm start`
   - **Environment:** `Node`
5. Деплой! URL будет `https://dominogame.onrender.com`

## 🤖 Настройка Telegram Bot

1. Откройте [@BotFather](https://t.me/BotFather)
2. Создайте бота: `/newbot`
3. Настройте Mini App: `/newapp`
4. Укажите URL вашего Render деплоя
5. Готово!

## 🎮 Функции

- ♟️ **Классические правила домино** (до 4 игроков)
- 🎨 **Красивые кости домино** с точками
- 🃏 **Стол в стиле покера** с зелёным сукном
- 🔊 **Звуковые эффекты** (Web Audio API)
- 📳 **Вибрация** на мобильных
- 🎉 **Конфетти** при победе
- 🔍 **Поиск столов** в реальном времени
- ⚡ **Быстрая игра** — авто-присоединение
- 📊 **Счётчики** игроков, столов, игр
- 🔄 **Реал-тайм** обновления через Socket.IO

## 🛠 Технологии

- **Backend:** Node.js + Express + Socket.IO
- **Frontend:** Vanilla JS + CSS animations
- **Deploy:** Render.com
- **Integration:** Telegram WebApp SDK

## 🎯 Правила игры

- Каждый игрок получает 7 костей (при 2 игроках) или 6 (при 3-4)
- Первый ход — у кого самый высокий дубль
- Кость ставится соответствующим значением к открытому концу цепочки
- Нельзя поставить — берите из бонеярда
- Первый, кто выложил все кости — победитель
- Победитель получает сумму очков оставшихся костей соперников
